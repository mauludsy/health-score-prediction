# health-score-prediction
## About Dataset
The dataset used in this project is a synthetic dataset created to simulate health score prediction based on the following variables:
- Age
- Body Mass Index (BMI)
- Exercise_Frequency
- Diet_Quality
- Sleep_Hours
- Smoking_Status
- Alcohol_Consumption
- Health_Score

The dataset is not included in this repository to maintain repository cleanliness and simplicity.
